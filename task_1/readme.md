# Описание проекта

Этот проект представляет собой пайплайн для обработки данных в папке `data`. Пайплайн состоит из нескольких этапов, каждый из которых выполняет определенную операцию над данными.

## Этапы пайплайна

1. **Чтение данных**: Данные читаются из файла `data/input.csv` с использованием библиотеки pandas.
2. **Обработка данных**: Данные обрабатываются с помощью функции `process_data`, которая выполняет следующие операции:
   - Удаление дубликатов
   - Заполнение пропущенных значений
   - Очистка текстовых колонок от нежелательных символов
3. **Преобразование данных**: Данные преобразуются с помощью функции `transform_data`, которая выполняет следующие операции:
   - Создание новой колонки на основе существующих
   - Фильтрация данных по определенным условиям
4. **Запись результатов**: Обработанные и преобразованные данные записываются в файл `data/output.csv` с использованием библиотеки pandas.

## Использование пайплайна

Для запуска пайплайна необходимо выполнить скрипт `pipeline.py`. Скрипт будет автоматически выполнять все этапы, описанные выше, и записывать результат в файл `data/output.csv`.
